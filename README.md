# 🧮 Calculadora Java - Proyecto Colaborativo

Este proyecto es una aplicación de consola desarrollada en Java que funciona como una calculadora básica. Ha sido creado como parte de la Actividad 4.1 de Control de Versiones del módulo de Entornos de Desarrollo (1º DAM).

## ⚙️ Funcionalidades

La calculadora permite al usuario realizar las siguientes operaciones matemáticas básicas:
* Suma: Suma de dos números.
* Resta: Resta de dos números.
* Multiplicación: Multiplica dos números.
* División: Divide dos números (con control para evitar la división por cero).

## 🛠️ Tecnologías y Herramientas

* Lenguaje: Java
* Control de Versiones: Git y GitHub
* Testing: Pruebas unitarias implementadas con JUnit.
* Integración Continua (CI): Automatización de las pruebas mediante GitHub Actions (los tests se ejecutan automáticamente con cada cambio en el código).

## 🌿 Flujo de Trabajo (Git)

Para el desarrollo de este proyecto hemos utilizado un flujo de trabajo basado en ramas:
1. main: Rama principal que contiene la versión final y estable del código.
2. rama-Noelia Cobacho: Rama donde se han desarrollado dos de las operaciones.
3. rama-Germán Ayala: Rama donde se han desarrollado las otras dos operaciones.

Una vez finalizadas y probadas las operaciones, ambas ramas personales se fusionaron (*merge*) en la rama `main` manteniendo el historial de cambios.
